register(boolean|string  $filename = false) : mixed

Enregistre le fichier dans le dossier de destination si il peut être uploadé.

Parameters

boolean|string $filename

Chemin du fichier. Si FALSE, la fonction retournera le contenu du fichier.

Returns

mixed —

Si le fichier ne peut être uploadé, retourne FALSE. Si $filename n'est pas fourni ou est à FALSE, retourne le contenu du fichier, sinon écrit le fichier dans le dossier de destination.
